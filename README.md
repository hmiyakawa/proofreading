# 文章構成ツール

このワークスペースには、入力文章を構成し、変換結果と変更点を同時に表示するブラウザベースのツールがあります。

## 使い方

### ローカルサーバーを起動する場合

1. ターミナルでこのフォルダーに移動します。
2. `npm install` を実行します。
3. 環境変数 `ANTHROPIC_API_KEY` に Claude API キーを設定します。
   - macOS/Linux: `export ANTHROPIC_API_KEY=あなたのキー`
4. `npm start` を実行します。
5. ブラウザで `http://localhost:3000` を開きます。

### ブラウザで使う

1. 左の入力欄に文章を入力します。
2. 右上の「変換結果」で構成済み文章を確認します。
3. 「Claudeで校正」ボタンを押すと、Claudeによる校正結果が取得されます。
4. 下段の「校正差分」で、入力文章とClaude校正後の違いを確認できます。

## ファイル

- `index.html` : 入力・出力・差分を表示するWebページ
